# 📊 市场数据功能使用指南

## 功能概述

市场数据功能可以在宠物收益率计算器页面顶部显示当前市场的参考价格，包括金杯最低价、宠物最低价和1金杯可兑换灵石数量。

---

## 🎯 功能特点

### 1. 实时显示
- ✅ 在计算器页面顶部醒目展示
- ✅ 渐变色面板设计，美观大方
- ✅ 三个数据卡片，清晰直观

### 2. 数据编辑
- ✅ 点击"编辑"按钮即可修改
- ✅ 表单验证，确保数据有效
- ✅ 保存/取消操作，灵活控制

### 3. 数据持久化
- ✅ 自动保存到浏览器 LocalStorage
- ✅ 刷新页面数据不丢失
- ✅ 用户端和管理端数据同步

---

## 📋 显示内容

### 1. 🏆 金杯最低价
- **单位**：元
- **说明**：当前市场上金杯的最低价格
- **用途**：计算收益率时的参考数据

### 2. 🐾 宠物最低价
- **单位**：金
- **说明**：当前市场上宠物的最低价格
- **用途**：了解市场入门门槛

### 3. 💎 1金杯兑换灵石
- **单位**：灵石
- **说明**：1个金杯可以兑换的灵石数量
- **用途**：收益率计算的关键参数

---

## 🔧 使用方法

### 查看市场数据

1. 打开用户端或管理端页面
2. 进入"🐾 宠物收益率"页面
3. 在页面顶部即可看到市场数据面板

### 编辑市场数据

1. **点击编辑按钮**
   - 在市场数据面板右上角点击"✏️ 编辑"按钮

2. **填写数据**
   - 金杯最低价（元）：输入当前金杯最低价格
   - 宠物最低价（金）：输入当前宠物最低价格
   - 1金杯可兑换灵石数量：输入兑换比例

3. **保存或取消**
   - 点击"💾 保存"：保存数据并返回显示模式
   - 点击"❌ 取消"：放弃修改并返回显示模式

### 更新市场数据

建议定期更新市场数据，以保持计算的准确性：
- 市场价格波动时及时更新
- 游戏版本更新后检查数据
- 每周至少更新一次

## ⚠️ 注意事项

### 1. 数据准确性
- 市场数据仅供参考
- 请根据实际情况填写
- 建议多方核实后再更新

### 2. 数据范围
- 所有数值必须为正数
- 金杯价格和宠物价格支持小数
- 兑换比例必须为整数

### 3. 数据备份
如需备份市场数据：
```javascript
// 在浏览器控制台执行
console.log(localStorage.getItem('marketData'));
// 复制输出的JSON字符串保存
```

### 4. 数据恢复
如需恢复市场数据：
```javascript
// 在浏览器控制台执行
localStorage.setItem('marketData', '{"goldCupPrice":10,"petPrice":100,"goldCupExchange":1000}');
// 刷新页面即可
```

## 🔍 常见问题

### Q1: 为什么显示"--"？
A: 首次使用时还没有设置市场数据，点击"编辑"按钮添加数据即可。

### Q2: 数据会同步到其他设备吗？
A: 不会。数据保存在本地浏览器，不同设备需要分别设置。

### Q3: 清除浏览器数据后会怎样？
A: 市场数据会被清除，需要重新设置。

### Q4: 可以设置多个市场价格吗？
A: 目前只支持保存一组数据，建议保存最常用的参考价格。

### Q5: 如何知道数据上次更新时间？
A: 数据中包含updateTime字段，但目前界面未显示。可在控制台查看。
