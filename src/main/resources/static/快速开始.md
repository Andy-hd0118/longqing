# 🎯 快速开始指南

## 页面说明

本系统包含两个独立页面：

### 👥 用户端（index.html）
- **访问方式**：直接打开 `index.html` 或点击“管理者入口”旁边的链接
- **功能**：查看公告、宠物收益率计算、浏览开发者数据（只读）
- **适用人群**：所有普通用户

### 🔐 管理端（admin.html）
- **访问方式**：从用户端点击“🔐 管理者入口”按钮，或直接打开 `admin.html`
- **功能**：发布公告、管理开发者数据、查看数据统计
- **适用人群**：网站管理员

## 第一步：本地测试（立即体验）

### 最简单的方法
直接双击 `index.html` 文件，在浏览器中打开即可使用！

### 推荐方法（使用本地服务器）
**Windows 用户：**
- 双击运行 `start.bat` 文件
- 然后在浏览器访问：http://localhost:8080

**Mac/Linux 用户：**
```bash
cd src/main/resources/static
python3 -m http.server 8080
```
然后访问：http://localhost:8080

## 第二步：部署到网络（分享给他人）

### 🚀 超简单方法 - Netlify Drop（30秒完成）

1. **打开网址**：https://app.netlify.com/drop

2. **拖拽文件夹**：把整个 `static` 文件夹拖到网页上

3. **获得网址**：几秒钟后你会得到一个网址，例如：
   ```
   https://your-site-name.netlify.app
   ```

4. **分享网址**：把这个网址发给任何人！
   - ✅ 手机可以访问
   - ✅ 电脑可以访问
   - ✅ 平板可以访问
   - ✅ 任何设备都能访问

## 功能说明

### 用户端功能

#### 📢 查看公告
- 浏览管理者发布的最新公告
- 支持普通、重要、紧急三种优先级显示
- 按时间倒序排列

#### 🐾 宠物收益率计算器
专业的宠物投资回报分析工具，支持三种模式：

**出租模式：**
- 输入宠物价格、金杯兑换比例、金杯单价、出租收益
- 自动计算收益率和回本天数

**发车模式：**
- 输入宠物价格、金杯兑换比例、金杯单价、每天发车收益
- 自动计算收益率和回本天数

**PVP模式：**
- 输入宠物价格、金杯单价、掉落金杯数
- 自动计算收益率和回本天数

**智能评级：**
- 🌟 优秀：收益率 ≥ 5%
- 👍 良好：收益率 2% - 5%
- ⚠️ 一般：收益率 < 2%

#### 💻 查看开发者数据
- 浏览管理者发布的数据
- 使用搜索和筛选功能
- 卡片式展示

### 管理端功能

#### 📢 公告管理
- 填写标题和内容
- 选择优先级（普通/重要/紧急）
- 点击“发布公告”
- 可以编辑和删除已发布的公告

#### 💻 数据管理
- 填写数据名称、分类、类型、值
- 点击“添加数据”
- 可以编辑和删除已发布的数据
- 实时查看数据统计信息

## 常见问题

**Q: 用户端和管理端有什么区别？**
A: 用户端只能查看数据，管理端可以发布、编辑、删除数据。

**Q: 如何从用户端进入管理端？**
A: 在用户端页面顶部导航栏，点击“🔐 管理者入口”按钮即可。

**Q: 数据会丢失吗？**
A: 数据保存在浏览器中，只要不清除浏览器数据就会一直存在。

**Q: 不同设备能看到相同的数据吗？**
A: 不可以。每个设备的数据是独立的。如果需要共享，可以将数据截图或复制分享。

**Q: 需要联网才能使用吗？**
A: 本地使用不需要联网。如果部署到网络后，访问者需要联网。

**Q: 可以自定义样式吗？**
A: 可以！修改 `css/style.css` 文件即可。

**Q: 如何更新内容？**
A: 修改对应的 HTML 或 JavaScript 文件，重新部署即可。
