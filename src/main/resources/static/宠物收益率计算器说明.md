# 🐾 宠物收益率计算器使用指南

## 功能概述

宠物收益率计算器可以帮助你快速计算不同模式下宠物的投资回报率，包括**出租模式**、**发车模式**和**PVP模式**三种计算方式。

---

## 📊 计算公式

### 1. 🏠 出租模式

```
收益率 = 出租收益灵石数量 / 1金杯可兑换灵石数量 × 金杯单价 / 宠物本体价格
```

**适用场景**：将宠物出租给其他玩家获得灵石收益

**示例**：
- 宠物价格：100金
- 1金杯可兑换：1000灵石
- 金杯单价：10元
- 出租收益：5000灵石

```
收益率 = 5000 / 1000 × 10 / 100 = 0.5 = 50%
回本天数 = 1 / 0.5 = 2天
```

---

### 2. 🚗 发车模式

```
收益率 = 每只宠物预计一天发车灵石收益 / 1金杯可兑换灵石数量 × 金杯单价 / 宠物本体价格
```

**适用场景**：使用宠物参与发车活动获得每日灵石收益

**示例**：
- 宠物价格：200金
- 1金杯可兑换：1000灵石
- 金杯单价：10元
- 每天发车收益：8000灵石

```
收益率 = 8000 / 1000 × 10 / 200 = 0.4 = 40%
回本天数 = 1 / 0.4 = 2.5天
```

---

### 3. ⚔️ PVP模式

```
收益率 = 掉落的金杯数 × 金杯价格 / 宠物本体价格
```

**适用场景**：通过PVP战斗获得金杯奖励

**示例**：
- 宠物价格：150金
- 金杯单价：10元
- 掉落金杯数：10个

```
收益率 = 10 × 10 / 150 = 0.667 = 66.7%
回本天数 = 1 / 0.667 ≈ 1.5天
```

---

## 🎯 使用步骤

### 第一步：填写基础信息

所有模式都需要填写以下三项：

1. **宠物本体价格（金）**
   - 输入你购买宠物的价格
   - 单位：金

2. **1金杯可兑换灵石数量**
   - 输入游戏内1个金杯可以兑换多少灵石
   - 例如：1000

3. **金杯单价（元）**
   - 输入金杯的市场价格
   - 单位：元

### 第二步：选择计算模式

点击对应的模式卡片：

- 🏠 **出租模式**：适合长期稳定收益
- 🚗 **发车模式**：适合日常活跃玩家
- ⚔️ **PVP模式**：适合喜欢战斗的玩家

### 第三步：填写模式专属数据

根据选择的模式，填写对应的数据：

**出租模式**：
- 出租收益灵石数量：每次出租能获得的灵石

**发车模式**：
- 每只宠物预计一天发车灵石收益：每天的灵石收入

**PVP模式**：
- 掉落的金杯数：预计能获得的金杯数量

### 第四步：点击计算

点击"计算收益率"按钮，系统会自动显示：

✅ **收益率百分比**：投资的回报率  
✅ **计算公式**：详细的计算过程  
✅ **回本天数**：多少天可以收回成本  
✅ **评级徽章**：优秀🌟 / 良好👍 / 一般⚠️

---

## 📈 评级标准

系统会根据收益率自动评级：

| 评级 | 收益率范围 | 说明 |
|------|-----------|------|
| 🌟 优秀 | ≥ 5% | 投资收益非常好，建议投资 |
| 👍 良好 | 2% - 5% | 投资收益不错，可以考虑 |
| ⚠️ 一般 | < 2% | 投资收益较低，谨慎投资 |

## 💡 使用技巧

### 1. 对比不同模式
可以分别计算三种模式的收益率，选择最优的投资方式。

### 2. 调整参数测试
修改不同的数值，观察收益率的变化，找到最佳投资点。

### 3. 考虑风险
- 出租模式：相对稳定，但可能有空置期
- 发车模式：需要日常活跃，收益较稳定
- PVP模式：波动较大，取决于战斗表现

### 4. 回本天数参考
- **1-3天**：非常快，强烈推荐
- **3-7天**：较快，值得考虑
- **7-14天**：一般，需谨慎
- **超过14天**：较慢，建议 reconsider

## ❓ 常见问题

### Q1: 为什么我的收益率是负数？
A: 检查输入的数据是否正确，确保所有数值都是正数。

### Q2: 回本天数显示"无法回本"是什么意思？
A: 当收益率为0或负数时，表示无法通过该方式回本。

### Q3: 如何判断哪个模式更好？
A: 比较三个模式的收益率和回本天数，选择收益率高且回本快的模式。

### Q4: 计算结果准确吗？
A: 计算基于你输入的数据，请确保数据的准确性。实际收益可能受市场波动影响。

### Q5: 可以保存计算结果吗？
A: 目前不支持保存，建议截图记录重要的计算结果。
