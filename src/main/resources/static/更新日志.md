# 🎉 宠物收益率计算器 - 新功能发布

## 📋 更新概述

本次更新为数据计算器新增了**宠物收益率计算**功能，支持三种不同的收益模式计算。

---

## ✨ 新增功能

### 🐾 宠物收益率计算器

一个专业的宠物投资回报分析工具，帮助玩家快速计算不同模式下的收益率。

#### 支持的三种模式

1. **🏠 出租模式**
   - 计算公式：收益率 = 出租收益灵石 / 金杯兑换灵石 × 金杯单价 / 宠物价格
   - 适合长期稳定收益的投资者

2. **🚗 发车模式**
   - 计算公式：收益率 = 每天发车灵石收益 / 金杯兑换灵石 × 金杯单价 / 宠物价格
   - 适合日常活跃玩家

3. **⚔️ PVP模式**
   - 计算公式：收益率 = 掉落金杯数 × 金杯价格 / 宠物价格
   - 适合喜欢战斗的玩家

## 🎯 使用指南

### 快速开始

1. **打开计算器**
   - 用户端：点击"🐾 宠物收益率"标签
   - 管理端：进入"🧮 计算器"页面，点击"🐾 宠物收益率"标签

2. **填写基础信息**
   - 宠物本体价格
   - 1金杯可兑换灵石数量
   - 金杯单价

3. **选择模式**
   - 点击对应的模式卡片

4. **填写模式数据**
   - 根据选择的模式填写相应数据

5. **查看结果**
   - 点击"计算收益率"按钮
   - 查看收益率、公式、回本天数和评级

请查看 `宠物收益率计算器说明.md` 文件
